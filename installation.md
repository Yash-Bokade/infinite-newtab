# Installation 

## NPM | BUN Based

> for installation of the plugin you must build the project

#### Steps

1. Clone the Repo using 

```
git colne https://github.com/Yash-Bokade/infinite-newtab.git
```

```
cd infinite-newtab
```
2. Install dependencies
  - NPM
```
npm i
```
  - BUN
```
bun i
```

3. Build
  - NPM
```
npm run build
```
  - BUN
```
bun run build
```

5. adding Extension
 - go to `chrome://extensions` in chrome and turn on `Developer Mode` in top right corner.
